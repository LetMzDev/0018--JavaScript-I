## Comandos Terminal
**Função** | **Descrição**
---|---
`node -version` | Verifica a versão atual do `nodeJS`
`node -v` | Verifica a versão atual do `nodeJS`
`node` | Abre o interpretador no `PowerShell`
`.exit` | Sair do modo `node` no terminal
`node arquivo.js` | Executar o arquivo
